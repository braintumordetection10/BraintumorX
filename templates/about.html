About BrainTumorX

BrainTumorX is an AI-powered clinical support system designed to aid radiologists in identifying and classifying brain tumors from MRI scans. It uses a hybrid deep learning architecture combining multiple CNN models and machine learning techniques.

The platform is optimized for academic research, clinical demonstration, and intelligent medical software development. It delivers fast predictions, confidence scoring, and transparent AI using Grad-CAM heatmaps.

Core Features

Multi-CNN Feature Fusion

Uses VGG16, ResNet50, and InceptionV3 extracted features for higher diagnostic accuracy.

PCA + LDA Reduction

Removes noise and extracts highly discriminative tumor features.

Hybrid Ensemble Classifier

Combination of SVM, KNN, and Random Forest provides reliable predictions.

Grad-CAM Explainability

Highlights tumor-specific regions for transparent and interpretable diagnostics.
